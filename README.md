## 📌 Overview
Perpustakaan Pribadi Digital adalah solusi sederhana untuk mengelola koleksi buku secara mandiri.

Tidak hanya sebagai tempat membaca, platform ini juga memungkinkan pengguna untuk:
- ✍️ Membuat buku sendiri  
- 📚 Menyimpan koleksi pribadi  
- 🗂️ Mengatur kategori buku  
- ☁️ Mengakses dari mana saja  

---

## ⚙️ Features

- 📚 **Manajemen Buku**  
  Tambah, edit, dan hapus buku dengan mudah  

- ✍️ **Buat Buku Sendiri**  
  Tulis dan simpan konten langsung dari web  

- 🗂️ **Kategori & Organisasi**  
  Kelompokkan buku sesuai kebutuhan  

- 🔍 **Pencarian Buku**  
  Temukan buku dengan cepat  

- ☁️ **Akses Online**  
  Bisa diakses kapan saja & di mana saja  

---

## 🔄 How It Works

1. User masuk ke web  
2. Menambahkan atau membuat buku  
3. Buku tersimpan dalam sistem  
4. Buku dapat dibaca, diedit, atau dikelola kembali  

---

## 💡 Benefits

- 🔐 Data milik pribadi (full control)  
- ⚡ Mudah dan ringan digunakan  
- 📱 Bisa diakses dari HP maupun laptop  
- 🧠 Cocok untuk belajar & menulis  
- 🚀 Bisa dikembangkan lebih lanjut
